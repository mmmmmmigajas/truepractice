# 📸 Práctica de Velocidad en JavaScript

Este es mi primer ejercicio práctico usando **operadores ternarios** en JavaScript para evaluar rangos de velocidad.

## 🚀 Lo que aprendí
* Cómo usar el operador ternario encadenado.
* Configuración de Git y GitHub desde cero.
* Corrección de errores de sintaxis en la terminal.

## 🛠️ Cómo ejecutarlo
Solo necesitas abrir el archivo `cam.js` en VS Code y ejecutarlo con Node.js o pegarlo en la consola de tu navegador.
